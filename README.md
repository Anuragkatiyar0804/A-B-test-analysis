# A/B Test Analysis: Product Feature Rollout
An end-to-end A/B test analysis project simulating a real product experiment (new onboarding/recommendation feature), built to demonstrate proper experimental design and statistical rigor — not just ran a t-test and got a p-value.

# Why this project
Built specifically to demonstrate skills required for Product Analyst / Data Analyst roles involving experimentation: designing tests correctly,validating experiment integrity before trusting results, and translating statistical output into a business decision — including catching a case where the pooled result would mislead you.
